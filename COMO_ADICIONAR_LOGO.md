# Como Adicionar o Logo

## Passo a Passo

1. **Coloque o arquivo do logo na pasta `public/`**
   - Nome do arquivo: `logo-frango-dourado.png`
   - Formatos aceitos: PNG, JPG, SVG
   - Tamanho recomendado: altura de 50-80px

2. **O logo aparecerá automaticamente no header do cardápio**

3. **Se o logo não aparecer:**
   - Verifique se o arquivo está em `public/logo-frango-dourado.png`
   - Verifique o nome do arquivo (deve ser exatamente `logo-frango-dourado.png`)
   - Recarregue a página

## Estrutura de Pastas

```
public/
  └── logo-frango-dourado.png  ← Coloque seu logo aqui
```

O sistema já está configurado para carregar o logo automaticamente!
