# Sons de Notificação

Esta pasta contém os arquivos de áudio para as notificações do sistema.

## Arquivos Necessários

Coloque os seguintes arquivos nesta pasta:

1. **cash-register.mp3** - Som de caixa registradora para novos pedidos
2. **bell.mp3** - Som de campainha para chamadas de atendente

## Formatos Aceitos

- **MP3** (recomendado) - Melhor compatibilidade e menor tamanho
- **WAV** - Alta qualidade, mas arquivos maiores
- **OGG** - Boa qualidade e compressão
- **AAC** - Boa qualidade, compatível com navegadores modernos

## Recomendações

- **Duração**: 0.5 a 2 segundos (sons curtos são mais eficazes)
- **Volume**: Normalizado (não muito alto nem muito baixo)
- **Qualidade**: 128kbps para MP3 é suficiente
- **Tamanho**: Tente manter abaixo de 100KB por arquivo

## Como Adicionar

1. Baixe ou crie os arquivos de som
2. Renomeie-os para:
   - `cash-register.mp3` (para pedidos)
   - `bell.mp3` (para chamadas)
3. Coloque-os nesta pasta (`public/sounds/`)
4. O sistema detectará automaticamente e usará os arquivos
5. Se os arquivos não existirem, o sistema usará sons gerados programaticamente

## Teste

Após adicionar os arquivos, use os botões "Testar Som" no dashboard para verificar se estão funcionando.
